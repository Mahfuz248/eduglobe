# **App Name**: EduGlobe

## Core Features:

- Dual-Language Support: Implement a language toggle (English/Bangla) for all site content, switching between languages for titles, descriptions, and result headers.
- Student Result Portal: Create a section for students to enter their ID and fetch their details (name, class, roll, subject-wise grades, pass/fail status).
- Result Data Lookup: Implement a JavaScript-based data structure for storing student results to perform quick lookups.
- Photo Gallery: Design a grid-style photo gallery to showcase campus life and events.
- Contact Section: Integrate a Google Map, school address, and a functional contact form.
- Homepage Slider: Implement a photo slider for the homepage with news announcements and school history sections.

## Style Guidelines:

- Primary color: Navy blue (#2E3192) to convey trust and professionalism.
- Background color: Light gray (#F5F7FA), providing a clean, modern backdrop.
- Accent color: Yellow-gold (#FFC107) to highlight key elements and calls to action.
- Headline font: 'Space Grotesk' (sans-serif) for headlines, offering a modern and techy feel.
- Body font: 'Inter' (sans-serif) for body text, providing a neutral and readable style. Ensure compatibility and readability for Bangla script by using 'Hind Siliguri' for the Bangla version of the site.
- Use modern, flat icons for navigation and key features.
- Mobile-first, responsive layout for seamless access on smartphones and desktops.
